# etcdWp

**etcdWp**是一款免费且支持Macos桌面的etcd客户端桌面 **(Mac Version >= 11.0)**。支持对etcd单机版和集群版键值存储做增删改查等操作。
同时，也增加对etcd集群资源和成员操作。

# 功能支持

- [X] 支持增加,删除,修改,查看。
- [X] 支持对键值批量导出,批量导入。
- [X] 支持对租约增加,删除,修改,查看。
- [X] 支持对集群成员增加,删除,修改,查看。
- [X] 支持对用户增加,删除,修改,查看。
- [X] 支持显示对集群资源,leader状态。
- [X] 支持键值平铺及树形结构展示。
- [X] 增加对etcd操作日志显示。
